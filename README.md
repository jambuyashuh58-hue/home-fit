# HomeFit Next.js App

This zip contains a minimal Next.js 14 App Router project that renders your `components/homefit-final.jsx` as the root page.

## Run locally
```bash
npm i
npm run dev
```

## Deploy to Vercel
- Push the contents of this folder to your GitHub repo (as the **repo root**).
- On Vercel, no special settings are required; it will auto-detect Next.js and deploy `/` from `app/page.jsx`.
